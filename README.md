Drafty ist ein moderner, intuitiver **No-Code-Web-Builder** direkt für den Browser. Mit Drafty kannst du per Drag-and-Drop responsive Websites, Landingpages und Layouts im Handumdrehen entwerfen, stylen und exportieren.

---

## ✨ Features

* 📁 **Seiten-Manager:** Verwalte mehrere HTML-Seiten direkt in deinem Workspace. Erstelle neue Seiten oder lösche sie mit einem Klick.
* 🧱 **Drag-and-Drop Bausteine:** Große Auswahl an Komponenten (Überschriften, Textblöcke, Navigationsleisten, mehrspaltige Layouts, Karten, Zitate, Formulare).
* ⭐ **Favoriten & Häufig genutzt:** Markiere deine Lieblingskomponenten mit einem Stern, um noch schneller auf sie zuzugreifen.
* 🎨 **Intelligentes Design-Thema:**
  * Generiere vollautomatisch passende Farbharmonien.
  * Nutze das integrierte **Raster & Lineal** für pixelgenaues Arbeiten.
  * Vertikale Abstände lassen sich mit einem Klick vereinheitlichen.
* 🔍 **SEO & Custom Code:** Pflege Meta-Tags direkt ein oder erweitere dein Projekt mit eigenem CSS und JavaScript.
* 📸 **Export-Optionen:** Lade dein fertiges Projekt als sauberen HTML-Code herunter oder erstelle direkt einen hochauflösenden Screenshot deines Designs.
